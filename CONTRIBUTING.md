# Внесок у проєкт

Дякуємо за інтерес до проєкту! Ми раді будь-яким внескам.

## Як внести свій внесок

1. **Fork** репозиторій
2. Створіть **feature branch** (`git checkout -b feature/AmazingFeature`)
3. Зробіть **commit** змін (`git commit -m 'Add some AmazingFeature'`)
4. **Push** до branch (`git push origin feature/AmazingFeature`)
5. Відкрийте **Pull Request**

## Стандарти коду

- Дотримуйтесь існуючого стилю коду
- Додавайте коментарі для складних частин коду
- Переконайтеся, що ваш код проходить linting
- Додавайте тести для нових функцій (якщо можливо)

## Структура комітів

Використовуйте чіткі та описові повідомлення комітів:

- `feat:` для нових функцій
- `fix:` для виправлень помилок
- `docs:` для змін у документації
- `style:` для форматування коду
- `refactor:` для рефакторингу
- `test:` для додавання тестів
- `chore:` для змін у конфігурації

## Питання?

Якщо у вас є питання, відкрийте issue або зверніться до maintainer.
