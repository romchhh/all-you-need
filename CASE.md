# All You Need — комерційний кейс

## Що зроблено

**Маркетплейс у Telegram**: бот + міні-додаток для продажу та покупки товарів без виходу з месенджера.

## Задача

Дати бізнесу готовий інструмент: каталог оголошень, профілі продавців, оплата та адмінка — все в одному каналі комунікації з клієнтами (Telegram).

## Рішення

- **Бот (Python, aiogram)** — створення оголошень, модерація, розсилки, реферали, інтеграція з платіжною системою (Monobank).
- **Міні-додаток (Next.js, TypeScript)** — каталог, фільтри, обране, рекомендації, оптимізація фото (WebP), профілі користувачів.
- **Спільна БД (Prisma + SQLite)** — одна схема для бота та веб-інтерфейсу, міграції, узгоджений стан даних.

## Результат

Повноцінний маркетплейс: від оголошення до оплати та статистики. Користувач працює в Telegram, власник — через бота та веб. Мультимовність (UA/RU), реферальна система, платні пакети та просування оголошень.

**Стек:** Python, aiogram 3, Next.js 16, TypeScript, Prisma, Tailwind, Monobank API, Telegram WebApp.
